# Actividades de Análisis de Datos

Repositorio que contiene distintas actividades realizadas para la materia **Fundamentos de Big Data**, enfocadas en análisis y visualización de datos usando Python.

---

## Tecnologías utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit
- Jupyter Notebook

---

## Actividades

### Actividad 2 – Dashboard de Análisis de Datos

En esta actividad se desarrolló un **dashboard interactivo** para visualizar un dataset mediante:

- Vista previa de la tabla
- Gráfica de barras
- Gráfica de dispersión
- Heatmap
- Matriz de correlación limpia

El dashboard fue construido utilizando **Streamlit** y librerías de visualización en Python.

---

## Cómo ejecutar el dashboard

Clonar el repositorio:
```bash
git clone https://github.com/EdgarAntonioTorres/Fundamentos-para-big-data

```

Instalar dependencias:

```bash
pip install streamlit pandas matplotlib seaborn

```

Ejecutar el dashboard:

```bash
python -m streamlit run dashboard.py
```
